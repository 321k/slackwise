SlackWise

Manage your TransferWise account from Slack.

Getting started

To start using the Slack bot, make sure you first click add to Slack. Then, use the /transferwise command from within Slack to set up the TransferWise integration. Once you are ready with the set-up, try one of the commands below.

Slash commands

Balances

/balances shows your TransferWise account balances.

Latest transfers

Use /latest to see the lastest transfers and activity.

Pay someone

/pay email amount currency creates a link that lets you pay someone.

Change home currency

/home-currency currency changes your home currency to any allowed currency.

Switch between profiles

If you have a business profile, you can use this /switch-profile to switch between your personal and business profile.

Give feedback

/transferwise-bot-feedback feedback.
